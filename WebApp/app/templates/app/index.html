OUR MISSION

We implement a convolutional neural network (CNN) model framework and Tansefer Learing concept , image per-processing, and set parameters to train the model for desktop application and web application. As a result the desktop application and web application developed an effective approach to detect brain tumors using MRI image to aid in making quick, efficient, and precise decisions. The prediction accuracy was used to assess performance. In the testing data, the algorithm outperformed the current conventional approaches for detecting brain Tumors and achieved excellent accuracy.
